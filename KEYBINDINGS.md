# Vollständige Tastenbelegung Neo-QWERTZ Ebene 3

Halte **Caps Lock** gedrückt und drücke die entsprechende Taste.

## Zahlenreihe

| Taste | Ebene 3 | Unicode/Name |
|-------|---------|--------------|
| 1 | ° | Grad-Zeichen |
| 2 | § | Paragraph |
| 3 | ℓ | Liter-Zeichen |
| 4 | » | Guillemets rechts |
| 5 | « | Guillemets links |
| 6 | $ | Dollar |
| 7 | € | Euro |
| 8 | „ | Anführung unten |
| 9 | " | Anführung links |
| 0 | " | Anführung rechts |
| ß | - | Minus (KP) |

## Navigation & Pfeiltasten

Diese Funktionen sind im Neo-Layout vorgesehen, werden aber in dieser Workaround-Version 
möglicherweise nicht alle unterstützt. Die Hauptfunktion liegt auf den Sonderzeichen.

## Hinweise

- Die Belegung entspricht weitgehend der Neo-Layout Ebene 3
- Fokus liegt auf den wichtigsten Sonderzeichen für Programmierung
- Griechische Buchstaben verfügbar für mathematische/wissenschaftliche Texte
- Caps Lock muss **gedrückt gehalten** werden (kein Toggle)
